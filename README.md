<p class=MsoNormal style='line-height:normal'><span style='font-size:12.0pt;
font-family:"Times New Roman","serif"'>A visualization exploration of a data
set containing 1157 baseball players and their corresponding handedness, height,
weight, home run counts and batting average.  We will examine how said physical
traits can influence a player’s performance.  </span></p>

<p class=MsoNormal style='line-height:normal'><b><span style='font-size:24.0pt;
font-family:"Times New Roman","serif"'>Design</span></b></p>

<p class=MsoNormal style='line-height:normal'><span style='font-size:12.0pt;
font-family:"Times New Roman","serif"'>The initial design phase included
sketching/developing various visualization techniques including bubble charts,
pie charts and area charts.  After trial and error, it was decided to proceed
with using bar plots with an <br>
an additional over-laid line series.  The bar plots were very useful in
creating the actual visualization of physical characteristics in relation to
performance.</span></p>
